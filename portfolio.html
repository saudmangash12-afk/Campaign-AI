🚀 Campaign AI

Marketing Intelligence Simulator

Portfolio Project

Project Overview

Campaign AI is a marketing intelligence simulator designed to analyze advertising performance across multiple social media platforms and recommend the most effective channel based on campaign goals and audience data.

Core Features

✔ Multi-platform analysis
✔ Audience age targeting
✔ Campaign type modeling
✔ KPI calculations
✔ Performance ranking
✔ Word report export

Platforms Included

TikTok • Instagram • Snapchat • Facebook • LinkedIn

Metrics Used

CTR • CPM • Conversion Rate • ROI • CPA

Business Value

The system simulates how marketers compare platforms, evaluate campaign performance, and make data-driven budget allocation decisions.

Skills Demonstrated

✔ Marketing Analytics
✔ Data Visualization
✔ JavaScript Logic
✔ KPI Optimization
